
# Adam Erler SAT and School Data Analysis

## Overview
This project focuses on exploring the relationship between educational data and two key outcomes: SAT performance and college attendance among students. It delves into various factors, including demographic information, and analyzes their impact on these outcomes.

## Features
- **Data Preprocessing**: The project includes a comprehensive process for preparing the data, with a focus on the control schools have over certain values.
- **Feature Selection**: Many features are carefully selected or excluded based on relevance and impact on student performance. This includes descriptive information about schools and measurements for lower grades.

## Milestones
- **Milestone 4 Overview**: Describes the detailed process for preparing the data.
- **Milestone 2 and 3**: Focuses on the relationship between student demographics and their performance on the SAT, as well as college attendance.

## Getting Started

### Prerequisites

- Required Python libraries (e.g., pandas, numpy, matplotlib) can be installed via `pip`.

### Installation
1. Clone the repo:
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
2. Install the required Python packages:
   ```sh
   pip install -r requirements.txt
   ```

### Usage
Run the Jupyter Notebook to view the analysis:
```sh

```
